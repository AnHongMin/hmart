1. 공통 유틸 구현
	- Class : com/hmart/common/
	- Js : web/js/common/
	
2. 모뉼  구현 : Class
	- com/hmart/{모뉼명}/
	ex) com/hmart/sample/

3. 모듈 구현 : Vm
	- web/vm/{모듈명}/
	ex) web/vm/sample/

4. 모듈 구현 : Js
	- web/js/{모듈명}/
	ex) web/js/sample/
	
5. 모듈 맵핑 설정
	- com/hmart/config/springmvc/controller.xml
	- com/hmart/config/sql-map-config.xml
	- com/hmart/sqlmap/Sample.xml
